
# Symbolic Reinforcement Learning for Solving Second Kind of Fredholm Integral Equations

Author: Saghar Khalilpour
M.Sc. Thesis Project — Shahid Beheshti University
2025

---

## Overview

This project proposes a grammar-guided symbolic reinforcement learning framework using Deep Q-Networks (DQN) to automatically discover solutions for Fredholm integral equations of the second kind.

The framework integrates symbolic expression generation, reinforcement learning, and physics-informed optimization to efficiently approximate analytical solutions.

## Methodology

The proposed framework combines:

* Grammar-guided symbolic expression generation
* Deep Q-Network (DQN) optimization
* Physics-informed residual minimization
* Automated equation discovery
* Reward-based convergence evaluation

The implementation is developed using PyTorch and scientific computing libraries.

---

## Results

The framework demonstrates convergence in discovering symbolic approximations for Fredholm integral equations using reinforcement learning-based optimization.
